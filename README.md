# 🎨 Siggy PFP Generator

Buat Profile Picture (PFP) unik dan keren dengan Siggy PFP Generator!

![Siggy PFP Generator](assets/images/preview.png)

## ✨ Fitur

- 🎨 **Base Warna Tubuh** - Pilih warna dasar karakter
- 👕 **Outfit** - Berbagai pilihan pakaian
- 🌄 **Background** - Solid, gradient, atau pattern
- 🏷️ **Logo** - Tambahkan logo dan teks custom
- 👓 **Accessories** - Kacamata, topi, headphone, dll
- 👁️ **Mata** - Berbagai style mata dengan custom color
- 👃 **Hidung** - Multiple nose styles
- ⚙️ **Opsi Tambahan** - Shadow, Outline, Mirror Mode
- 🎲 **Randomize** - Generate kombinasi acak
- 📥 **Download** - Export sebagai PNG
